# PanWatch(盯盘侠)功能总览

> 自托管 AI 股票助手,支持 A股 / 港股 / 美股。后端 Python(FastAPI + SQLite),前端 React(Vite + Tailwind + shadcn/ui),JWT 登录认证。
> 更新日期:2026-07-16

## 1. 持仓与自选管理

- **多账户持仓**:录入真实持仓(成本/数量/可卖数量),自动算总盈亏、今日盈亏、总市值、可用资金、总资产、仓位占比(高仓位/满仓预警)
- **自选关注池**:持仓 / 关注两个标签页,行情实时刷新(可配刷新频率)
- **大盘情绪卡**:0–100 情绪评分(冰点/偏冷/中性/偏暖/亢奋),合成自涨跌家数(50%)+ 涨跌停强度(25%)+ 两市主力净流入(25%)
- **板块主力资金卡**:行业板块今日主力净流入/流出 Top3,腾讯行情为主数据源、东财(akshare)兜底,后端缓存 5 分钟
- **个股洞察**:K线摘要评分、AI 深度分析弹窗、资金流向(超大单/大单/中单/小单四档)

## 2. 底仓 VWAP 做T盯盘

- **双向做T**:正T(低吸→反弹卖出)与倒T(高抛→回落买回),仅监控 A 股持仓并提醒,不自动交易
- **评分体系**(满分100,默认≥70触发):趋势(+20)、近支撑/压力(+20)、VWAP偏离(+15,门槛按ATR自适应)、止跌/滞涨反转(+20)、数据覆盖(+10)、盈亏比(+15);正T/倒T双向评分和指标标签(灰=未得分)同屏展示
- **闸门保护**:MA20 斜率趋势闸门(±2%)、涨跌停闸门(涨停不低吸/强势不高抛、跌停不接刀)、止损距离上限(ATR自适应)
- **状态机流程**:信号→确认成交(录实际价格数量)→盯对侧点位→平仓摊低成本,支持手动录入两腿、每日轮数上限、轮间冷却
- **离场方式**:固定价 / 跟踪止盈(trail) / 价格或评分任一
- 10 秒级分钟扫描调度,参数全部界面可调

## 3. 策略体系与信号

- **策略目录**(10个内置):趋势延续、MACD金叉、放量突破、回踩确认、Price Action、底仓VWAP做T、超跌反弹、Agent建议、市场扫描、**突破有效性**
- **信号快照**:每策略每日生成 `StrategySignalRun`(买/卖动作、入场区间、止损、目标价、评分)
- **后验评估**:按 1/3/5/10 日窗口评估信号质量,策略自动调权重、生成排名
- **入场候选榜**:多策略融合的候选股列表(量价/涨停/前高数据补齐)

### 3.1 突破有效性策略(v3.0 文档实现)

- **纯规则状态机**:60日前高定位(前高年龄≥5日)、收盘突破+0.5%缓冲、上升趋势背景(四线多头+MA60向上),锚点(D0/G0/G_SUPPORT/L0)一经冻结不漂移
- **四条确认路径**:A 放量突破站稳 / B 后置放量 / C 缩量回踩 / D 供给收缩;价格接受(≥60%天数站上G0)+ 最小确认3日
- **失效检查**(优先级最高,永久失效):收盘<G_SUPPORT、<L0、连续3日<G0、放量破位、跌停破位(含涨跌停取整+容差口径)
- **八态输出**:not_in_pool / observed / pending / valid_active / extended / failed / expired / invalidated
- **全市场扫描**:沪深主板约 3100 只(剔除创业板/科创板/北交所/ST/退市),每交易日 15:20 自动扫描 + 持仓失效复评
- **API**:`/api/breakout/scan`(手动扫描)、`/diagnose/{symbol}`(单股诊断)、`/signals`、`/monitor`

## 4. 模拟盘(Paper Trading)

- **自动交易闭环**:盘中每 60 秒扫描——现价进入信号入场区间自动建仓(默认单笔 5% 资金,A股整百股),止损/止盈/信号反转自动平仓
- **策略选择**:全部 / 自定义勾选 / Top-N(按策略排名),选择在 SQL 层过滤建仓与盘前计划口径一致
- **信号时效**:active 信号超过 5 天不再触发建仓(防陈旧入场区间)
- **市场资金分配**:CN/HK/US 按比例分池,比例 0 即不投入
- **成本模拟**:手续费/印花税/滑点按策略配置
- **统计**:总盈亏、胜率、最大回撤、资金曲线,每笔交易记录退出原因(止损/止盈/信号反转)
- **通知**(默认关,界面开启):实时建仓/平仓、盘前计划(09:00)、日终摘要(15:30)

## 5. AI Agents

| Agent | 说明 |
|---|---|
| 盘前分析(premarket_outlook) | 开盘前市场展望 |
| 盘中监测(intraday_monitor) | 交易时段异动监控提醒 |
| 收盘复盘(daily_report) | 收盘后持仓复盘报告 |
| 新闻速递(news_digest) | 新闻抓取+AI摘要(能力型,被其他agent调用) |
| 技术分析(chart_analyst) | K线技术面分析(能力型) |
| TradingAgents 深度分析 | 多角色协作的个股深度研究 |

- Agent 均可配置调度(cron)、模型、提示词模板(`prompts/` 每 agent 一个文件),历史运行记录可查
- AI 服务:OpenAI 兼容 API,模型/密钥/BaseURL 界面可配

## 6. 市场数据与资讯

- **行情/K线**:多数据源编排(efinance/akshare/腾讯/东财),提供方失败自动切换,K线缓存(复权变化自动重建)
- **市场事件**:涨跌停、异动等事件流
- **板块池**:按生产资料体系六大分类(金融/资源/能源转化/通道/被消费的制造/产业链主题)预置全量大板块地图;左侧分类树+右侧K线详情,支持 120日/1年/3年/5年 周期(3年以上自动周K);板块K线上可打事件标注(政策/产业/业绩/宏观/标注,重要事件宽视野仍显示),点击时间轴条目定位K线;池内板块可 pin 为重点关注(沿用8个上限)
- **板块估值分位**:行业板块显示 PE/PB 处于近3年/5年历史百分位(<20%低估绿 / 20-80%合理 / >80%高估红),头部徽章+左树圆点提示"高估/低谷";数据源申万一级行业日频估值(腾讯板块码 pt01<申万码> 免映射),收盘后自动增量、冷启动自动回填约5年历史;概念板块无申万口径显示"—"
- **资讯流**:多来源新闻聚合、主题快照、AI 排序
- **市场扫描快照**:全市场宽度/活跃度,用于市场状态(bullish/neutral/bearish)分类
- **数据源管理页**:各数据源健康状态与启停

## 7. 选股与发现

- **选股器(Screener)**:条件筛选,运行结果留存
- **机会(Opportunities)**:候选机会自动刷新(每日三次)
- **发现(Discover)**:股票建议池(带反馈闭环,建议 14 天过期)
- **回测(Backtests)**:策略历史回测引擎

## 8. 提醒与通知

- **价格提醒**:目标价/涨跌幅条件,60 秒轮询,命中记录留存
- **通知渠道**:Telegram、企业微信、钉钉、Bark、Webhook(Apprise),多渠道并发、可设默认渠道
- **去重与静默策略**:通知幂等窗口防重复轰炸
- **交易规则(TradeRules)**:风控参数(默认止损/止盈比例等)集中配置

## 9. 系统与运维

- **认证**:JWT 登录(账号密码 `.env` 配置)
- **设置中心**:AI服务、通知、股票链接平台(雪球等)、数据清理保留期等全部界面化
- **数据管理**:每日 4:00 housekeeping 按保留期清理(新闻45天/Agent运行90天/信号180天等,可配);日志定量封顶(总12万条);迁移前自动备份 SQLite
- **日志**:结构化入库,按 agent/事件/trace 检索
- **部署**:
  - Docker:`docker compose up -d --build`(源码构建)或 `bash scripts/deploy-local.sh`(默认 18123 端口,`PANWATCH_PORT` 可覆盖)
  - 裸机:`bash scripts/deploy-native.sh`(venv+前端构建+uvicorn,默认 18123;start/stop/restart/status/logs)
- **测试与CI**:380+ pytest 单测(中文用例名),pre-push 钩子跑测试,GitHub Actions 发布前测试拦截

## 10. 技术架构速览

```
server.py                 入口:DB初始化、Agent/数据源注册、调度器(APScheduler×6)
src/web/                  FastAPI 路由(/api/*)、SQLAlchemy 模型、JWT
src/agents/               AI Agent 业务逻辑
src/collectors/           无状态数据采集(行情/K线/新闻/资金流)
src/core/                 核心:AI客户端、通知、调度、模拟盘引擎、做T引擎、
                          策略引擎/目录/池、突破有效性扫描器、市场情绪、数据清理
src/core/signals/         纯信号计算(做T VWAP、突破有效性等)
prompts/                  Agent 提示词模板
frontend/                 React SPA(pnpm monorepo:api/base-ui/biz-ui)
data/                     SQLite + 运行时文件
```

API 响应统一包裹 `{code, data, message}`;所有用户界面文案为中文。
